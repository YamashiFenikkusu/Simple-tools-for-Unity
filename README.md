<h1>SIMPLE TOOLS FOR UNITY</h1>
This repository contain few simple tools to add in an Unity project. For the moment, just two are present: a procedural decors instancer and a audio listener to texture converter. I'll add more scripts later.</br>

<centre>____________________________________</centre>
<h2>Procedural Decor Assets</h2>
This simple tool place random "TransformToPlace" prefab on a nav mesh or on the actual point of view in the scene view of Unity. After placing, you can generate a basic (but nice and natural) staging.
<img alt="How to use procedural decor assets" src="https://github.com/YamashiFenikkusu/Simple-tools-for-Unity/blob/main/ProceduralDecorsAssets/PDA.png"></br>

<h2>Audio Listener to Texture Converter</h2>
This short script can convert an audio listener to 1D texture and apply it on a material.
